# nanairo_uncen
支持日文版和汉化版  
请确认  
日文版的exe（nanairo.exe）  
汉化版的exe（nanairo_CHS_Packed.exe）    
是否存在游戏目录下，如果不存在则报错。  

日文版  
下载nanairo_uncen.7z，  
然后解压后得到 nanairo_uncen.exe 放入游戏目录下运行即可  

汉化版  
下载nanairo_uncen_cn.7z，  
然后解压后得到 nanairo_uncen.exe, nanairo_uncen.pck 放入游戏目录下运行即可  
（本补丁内，不含任何汉化组的相关文件，本补丁与汉化组无任何关系）
